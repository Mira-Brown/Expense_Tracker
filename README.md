# Expense tracker
This expense tracking system helps users manage their personal finances. 
The system will allow users to record expenses, categorize, edit and delete files.
 
# Project Structure
Data- Contains sample csv files for data manipulation and persistent
Reports- contains generated report(to be done)
expense_tracker.ipynb - project source code

# How to run this project
Clone this repository
Open project in vscode
Open expense tracker.ipynb
Run all cells
Based on the menu options select a given choice
